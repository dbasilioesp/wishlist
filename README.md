# Wishlist

## Pre-requisites

- NodeJS >=22
- pnpm

## Installing

To install pnpm:

```cmd
npm install -g pnpm
```

To run API in ExpressJS:

```cmd
cd api-expressjs
pnpm install
pnpm run dev
```

To run App Vue:

```cmd
cd app-vue
pnpm install
pnpm run dev
```

## Storybook

To run storybook inside app vue:

```cmd
pnpm run storybook
```

## Tests

In ExpressJS:

```cmd
pnpm run test
```

In Vue:

```cmd
pnpm run test:unit
```
